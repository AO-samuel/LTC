Learn To Code

Learn To Code(LTC) is a network of focused, passion filled youths with an interest in teaching and learning everything there is to know about programming.

Learn

LTC offers practical,active learning with well abled mentors who are ready to give their all to their students.With an ever growing community LTC is becoming one of the largest platforms for learning and teaching everything there is to know about programming in Africa.

LTC Communities include:

Web Development

Learn how to build and host a Responsive Website.

Python Programming

Learn how to use pyhton in various areas like Data Science, Robotics, Machine Learning e.t.c

C#/C++ Programming

Learn how to use C# and C++ to build awesome games and applications.
